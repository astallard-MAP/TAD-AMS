# Cash4Houses Portal

## 🌟 Overview
A premium, AI-powered platform for distressed property sellers in London, Hertfordshire, and Essex. This portal uses real-time market data and AI triggers to identify and convert leads into guaranteed cash sales.

## 🚀 Key Features
- **AI Investigation Logic**: Automated preliminary property analysis (Land Registry & Comparable Sales).
- **Andy the Property Buyer - AI Chatbot**: Decades of experience built into an agentic chat widget.
- **Daily Market Analysis**: AI-generated news insights focused on "distressed" market triggers (interest rates, repossessions, etc.).
- **Multi-Channel Notifications**: Real-time Firestore lead generation and automatic email notifications via Office 365 Exchange.

## 🛠️ Stack
- **Frontend**: Vite + Vanilla JS + CSS3
- **Cloud Infrastructure**: Firebase (Hosting, Cloud Functions, Firestore, Storage)
- **AI Engine**: Google Gemini Pro (Text generation and News analysis)
- **SMTP**: Microsoft Office 365 Exchange

## 📦 Deployment
The project is configured for Firebase Hosting with the `dist` directory.

### Production Build:
```bash
npm run build
```

### Deploy to Live:
```bash
firebase deploy
```
